Что делают функции описал в самом файле.
Логика самого скрипта была такая:
    Для epub:
        Использую библиотеку которая парсит epub файлы.
        С помощью функции get_metadata получаю метадату файла.
        Паршу метадату чтобы получить из нее нужные поляю
    Для FB2:
        FB2 это по сути xml файл, поэтому его можно парсить с помощью bs4.
        С помощью bs4 паршу файл и находу нужные поля.

Далее в зависимости от расширения файла программа вызывает один методов.
Программа принимает путь до файла как аргумент командной строки.
